**🌐 Personal Portfolio**

A modern and responsive personal portfolio website built using React.js, Vite, and Tailwind CSS to showcase my skills, projects, certifications, and professional journey.


**🚀 Features**

Responsive design for all devices
Dark and Light theme support
Interactive and modern UI
Skills showcase
Project portfolio section
Certifications section
Resume page
Contact page with social links
Smooth navigation and animations


**🛠️ Tech Stack**

React.js
Vite
Tailwind CSS
React Router DOM
Framer Motion
Lucide React
React Icons
